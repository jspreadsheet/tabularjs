TabularJS Demo

Universal Spreadsheet File Parser for JavaScript

About TabularJS

TabularJS is a standalone library that converts spreadsheet files into JSON format. It supports 16+ file formats including Excel, OpenDocument, CSV, HTML tables, and more.

For this demo we use Jspreadsheet Pro. For more information about Jspreadsheet Pro: https://jspreadsheet.com

Note: TabularJS is format-agnostic. Use the parsed output with any spreadsheet component, data visualization library, or custom application.

Supported Formats

.xlsx .xls .ods .csv .tsv .xml .html .slk .dif .dbf .wks ...and more
